# Mój SaaS AI

Aplikacja Next.js z App Router, TypeScript i Tailwind CSS.

## Wymagania

- Node.js 18.17 lub nowszy
- npm, yarn lub pnpm

## Instalacja

```bash
npm install
```

## Uruchomienie

Uruchom serwer deweloperski:

```bash
npm run dev
```

Otwórz [http://localhost:3000](http://localhost:3000) w przeglądarce.

## Dostępne skrypty

- `npm run dev` - Uruchamia serwer deweloperski
- `npm run build` - Buduje aplikację do produkcji
- `npm start` - Uruchamia produkcyjny serwer
- `npm run lint` - Uruchamia linter ESLint
